## Install

### [lazy.vim](https://github.com/folke/lazy.nvim)

```lua
{
    "martynasmuizys/tzfn.nvim",
    name = "tzfn"
    config = function()
        require("tzfn").setup({
            enable = {
                terminal = true,
            },
            styles = {
                transparency = true,
            },
        })
    end,
}
```
